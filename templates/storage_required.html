⚠ 尚未設定資料儲存

這個部署還不能用

目前跑在唯讀的 serverless 環境（例如 Vercel），而且沒有連結資料庫（KV）。 站點清單、管理員密碼這些資料沒地方存，所以還不能開始用。請先完成下面設定。

請完成這兩步

  1. 連結 KV 資料庫：到 Vercel 專案的 Storage 分頁 → Create Database → 選 Upstash for Redis → 建好後按 Connect 連到此專案。 （會自動帶入 KV_REST_API_URL / KV_REST_API_TOKEN，不用手動加。）
  2. 重新部署（最容易漏）：環境變數只對新部署生效。到 Deployments → 最新那筆右邊 ⋯ → Redeploy。

完成後重新整理這頁，就會進到設定管理員密碼的畫面。

💡 不想處理這些設定的話，用 Docker 部署最省事：一行指令、資料存本機資料夾、完全不用碰 KV。 詳見 README。
